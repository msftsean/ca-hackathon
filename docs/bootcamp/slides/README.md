# Boot Camp Slides

This directory is a placeholder for presentation slides to be used during the 47 Doors Boot Camp.

---

## For Coaches and Instructors

Slide decks are intentionally **not included** in this repository. Coaches should create or customize presentations based on:

- Audience technical level (beginner, intermediate, advanced)
- Time available for the boot camp
- Specific institutional context or use cases
- Which labs will be covered in the session

---

## Suggested Slide Decks

### 1. Opening Presentation (30-45 minutes)

**Topics to cover:**

- Welcome and introductions
- The "47 Front Doors" problem in higher education
- Overview of what participants will build
- Three-agent architecture introduction
- Technology stack overview (Azure OpenAI, FastAPI, React)
- Boot Camp schedule and logistics
- How to get help

### 2. Lab Kickoff Slides (5-10 minutes each)

Brief introductions before each lab section:

- **Lab 00**: Environment setup walkthrough
- **Lab 01**: AI agents and intent classification concepts
- **Lab 02**: MCP (Model Context Protocol) introduction
- **Lab 03**: Spec-driven development philosophy
- **Lab 04**: RAG (Retrieval-Augmented Generation) fundamentals
- **Lab 05**: Agent orchestration patterns
- **Lab 06**: Azure deployment with azd

### 3. Closing Presentation (15-20 minutes)

**Topics to cover:**

- Recap of what was built
- Demo of a complete solution
- Next steps and resources
- How to continue learning
- Feedback collection

---

## Slide Creation Tips

1. **Keep slides visual** - Use architecture diagrams over bullet points
2. **Include live demo checkpoints** - Show participants what success looks like
3. **Add troubleshooting slides** - Common errors and fixes for each lab
4. **Time estimates** - Include realistic timing for each section
5. **QR codes** - Link to documentation and resources

---

## Recommended Tools

- **PowerPoint** or **Google Slides** for traditional presentations
- **Marp** for Markdown-based slides
- **Reveal.js** for developer-friendly HTML slides
- **Excalidraw** for architecture diagrams
